PHP的EOF(heredoc) 使用说明

  1. 必须后接分号，否则编译不通过
  2. EOF只是我们习惯的写法，其实无论什么字符都可以，只要保证前后一样即可，但不能出现中文字符的。还有常用的有EOT EOD 。
  3. 结束标识符必须独占一行比如:
    echo<<<EOF
    王小明学习PHP，
    万丈龙我学渣，
    桐生战兔学霸。
    EOF;
  4. 开始标识可以带引号也可以不带，但是要注意，如果带了引号，双引号内可以解析变量，单引号内不会解析变量或者转义字符。即使需要对内容内嵌时，也不用加转义字符。它本身能对单双引号转义。

PHP常量

  1. 常量类似变量,但是常量一旦被定义就不能被更改撤销。在PHP中常量贯穿整个脚本，作用域为全局。命名规则和变量一样，但是没有$开头
  2. 设置PHP常量，语法:define(“常量名”，”常量值”，true/false),第三个参数是定义是否对大小写敏感，false是敏感，true是敏感，如果不写第三个参数则默认为false敏感
