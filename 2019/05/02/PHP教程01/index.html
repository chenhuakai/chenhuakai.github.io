PHP 是什么

  • PHP（中文名：“超文本预处理器”）是一种通用开源脚本语言。语法吸收了C语言、Java和Perl的特点，利于学习，使用广泛，主要适用于Web开发领域.

PHP 可以干什么

  • PHP属于后起之秀，吸收了java和c以及perl等语言优点。
  1. PC端网站开发。60%全球互联网网站采用php技术，80%国内互联网网站使用php开发。这些网站包含购物网站，政府企业网站，QQ空间，论坛博客等等。
  2. 移动端微网站开发—小程序。移动设备的普及为移动互联网的快速发展奠定了基础。手机淘宝网站，手机京东网站等等， 微信公众号应用中的微网站。 将来微网站、公众号和小程序肯定会取代APP的地位。我们离不开手机，手机离不开互联网，互联网离不开网站，小程序，网站离不开PHP。
  3. APP后台开发。APP就像传统电脑上的CS（Client/Server）软件一样，前几年我们做的最多的就是将客户端的软件变成浏览器上的软件，不用安装客户端打开浏览器就可以使用该软件，人们用着方便，并且也没有客户端也不需要人们来维护客户端程序。现在公司为了节约成本直接做手机网站不再做APP。
  4. 微信小程序取代APP，这句话不是我说的，这是事实。如果我是企业老总，我肯定直接B/S，节约成本，实在产品淘汰，就换手做其他产品，根本不想多花成本去维护，更新升级实在太快啦。也不心疼。小程序不容小觑啊，是小程序就是未来，所以，我多bb一句，好好去学JavaScript，好好去学！通过微信小程序可以做出适合不同手机设备的移动应用。 我们不再开发APP了，那么微信官方推荐后台开发语言为PHP。 因为微信官方推出的官方Demo中只有PHP。

开始PHP语法入门

PHP语法

  • PHP放在哪里？
  • PHP可以放在文档中的任何位置。PHP以<?php开始，以?>结束
  • PHP 文件的默认文件扩展名是 “.php”。 比如add.php,conn..php,index.php.PHP 文件通常包含HTML和PHP代码。
  • 和大部分语言一样，PHP以;为分隔符。来学习PHP的基本上都有c系列语言的基础，我不多赘述分隔符;的概念。
  • PHP注释中，//为单行注释，// PHP天下第一 星号//为多行注释。这个也和很多一样啊。来看一看代码

PHP变量

  1. 变量规则
  • 变量以$座位前缀，后面跟着变量的名称，如$user
  • 变量名必须以字母或者下划线字符开始,包涵字母数字下划线
  • 变量名中不能有空格，万万注意
  • 变量名是严格区分大小写的。$user和$USER是不一样的

    来示范一下，创建PHP变量

PHP是一门弱类型语言，在声明变量的时候不需要声明数据类型，它根据变量的值自动转换数据类型，这个很方便。

  1. 变量作用域
  • PHP四种不同作用域local global static parameter

  • 在函数外定义的的变量拥有全局作用域，全局可使用，若要在函数内引用全局变量，需要使用 global 关键字。在函数内拥有局部作用域，可以在函数内部被访问。

  • global 关键字
    PHP中将所有全局变量存储在$GLOBALS[index]数组中。index是保存变量的名称。$GLOBALS[index]数组可以在函数内部访问，也可以更新全局变量。比较一下两种方式的不同

  • Static作用域,在第一次声明变量时使用 static 关键字可以使变量永远不被初始化，永远保留之前的操作结果。
  • 参数作用域 参数通过调用代码将值传递给函数做为局部变量。是函数声明的一部分

  1. PHP echo 语句

2.PHP print语句和echo基本是一样的，他们的区别在于

  • echo可以输出一个或多个字符串，print 只允许输出一个字符串
  • echo输出的速度比print快，echo没有返回值，print有返回值1

  • print()打印一个值（它的参数），如果字符串成功显示则返回true，否则返回false

    其他都一样

    3.PHP print_r 语句

  • 和echo类似
  • print_r显示关于一个变量的易于理解的信息,如果是string\integer\float，将打印变量值本身。

  • 如果是array，将会按照某种格式显示键和元素。对象就像数组一样。
